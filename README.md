# Pong

A modern take on the classic Pong game built in Unity, featuring two game modes.

## Game Modes
- **2 Players** — Play against a friend locally
- **Player vs AI** — Play against a computer opponent

## Controls
| | Player 1 | Player 2 |
|---|---|---|
| Move Up | W | Up Arrow |
| Move Down | S | Down Arrow |

## Features
- Screen shake on ball collision
- Particle effects on hits
- Volume slider
- Ball speed increases with every bounce — the longer the rally, the faster it gets
- Win screen when a player reaches the max score

## How to Play
1. Go to [Releases](https://github.com/SiddharthSNair23/Pong/releases/tag/V1.0)
2. Download `Pong.Game.zip`
3. Extract the folder
4. Run `Pong.exe`
## Built With
- Unity 6
- C#
